# Dashboard IoT para Automação Residencial com ESP32

Este repositório contém o desenvolvimento de uma dashboard web criada como parte de um projeto acadêmico da Faculdade Engenheiro Salvador Arena, na disciplina de Eletrotécnica e Instalações Elétricas. O projeto tem como objetivo a construção de uma maquete de uma residência inteligente composta por cinco subsistemas de automação: irrigação inteligente, controle de acesso, abertura automática de portão, controle de temperatura e iluminação por presença.

A dashboard foi desenvolvida utilizando HTML e integrada a um microcontrolador ESP32 por meio de um endereço IP gerado na rede local, permitindo a comunicação entre a interface web e os dispositivos físicos da maquete. Sua principal finalidade é demonstrar conceitos de Internet das Coisas (IoT), possibilitando o monitoramento e o controle remoto dos sistemas implementados.

Atualmente, as funcionalidades disponíveis na dashboard estão relacionadas aos subsistemas de irrigação inteligente e abertura automática de portão, permitindo a interação em tempo real com os dispositivos conectados. O projeto busca aplicar conhecimentos de automação, eletrônica, redes e desenvolvimento web, proporcionando uma experiência prática na integração entre hardware e software.
